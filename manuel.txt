1) ouvrir ubuntu puis creer l'environnement conda TOOLS_env :

conda create --yes -n TOOLS_env -c bioconda -c conda-forge \
    python=3.10 \
    samtools=1.13 \
    minimap2=2.22 \
    hisat2=2.2.1 \
    subread=2.0.1 \
    sra-tools=3.2.1 \
    trinity=2.15.1

2) conda activate TOOLS_env

3) streamlit run script.py
